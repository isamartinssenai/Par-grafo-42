# 📱 Site Laravel

## 📌 Sobre o projeto

O nosso trabalho tem como objetivo o desenvolvimento do site Parágrafo 42, uma plataforma de livraria digital voltada à apresentação e comercialização de livros que estão famosos nas redes de maior influência. O projeto foi idealizado como parte das atividades de desenvolvimento de software, buscando aplicar, de forma prática, os conhecimentos adquiridos ao longo das aulas.

## 🎯 Objetivo (MVP)

* Realizar cadastro de usuários
* Autenticar usuário (login)
* Armazenar token localmente
* Consumir dados da API

## 🚀 Tecnologias utilizadas

* Laravel
* Java Scripit
* PHP
* Json
* Bootstrap

## 📲 Funcionalidades

* Tela de Login
* Tela de Cadastro
* Validação de Usuário
* Consumo de API (GET, POST, PUT, DELETE)
* Tela Cadastro de Autor
* Tela de Cadastro de livros

## ⚙️ Como executar o projeto

```bash

git clone https://github.com/isamartinssenai/Par-grafo-42.git

# Acessar pagina princial
php artisan serve

👉 Endpoints utilizados:

* POST /Login Usuário
* GET /Cadastro de Usuário
* POST /Cadastrar autor
* POST /Cadastrar livros
